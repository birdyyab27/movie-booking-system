#MOVIE BOOKING SYSTEM

##DESCRIPTION
THIS IS A C CONSOLE-BASED APPLICATION THAT ALLOWS USERS TO BOOK MOVIE TICKETS,SELECT SEAT TYPE,AND GENERATE A TICKET

##FEATURES
-USER INPUT FOR NAME AND AGE
-MOVIE SELECTION
-SEAT TYPE SELECTION
-BILLING SYSTEM

##HOW TO RUN
-COMPILE THE PROGRAM USING A C COMPILER
-RUN THE EXECUTABLE FILE
-FOLLOW THE ON-SCREEN INSTRUCTIONS

##LANGUAGE USED
- C PROGRAMMING
